# Rain-Adaptive Traffic Simulation

A simulation framework for optimizing traffic flow with rain-adaptive pedestrian priority using SUMO.

## Setup

1. Install SUMO: [https://sumo.dlr.de/docs/Downloads.php](https://sumo.dlr.de/docs/Downloads.php)
2. Set SUMO_HOME environment variable
3. Install dependencies: `pip install -r requirements.txt`
